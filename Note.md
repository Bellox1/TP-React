# Gestion des etat:
 Un etat est une variable ou un donnée qui peut changer à tout moment 
 et quand elle change l'interface se mets à jours automatiquement. En React ont utilise les useState 
 Valeur precedente est recuperer par l'operateur ... avc les UseStates
# EventListener
  OnChange qui permet de detecter si un elment à changer de valeur
  OnClick si un element a été cliqué
# Condition de vue
  Directement dans la vue, ont peut faire: { variable ? "Valeur si vrai" : "Valeur si faux"}
  On peut aussi faire ceci si ont as des ClassName à appliquer:
  { Variable &&   <p ClassName="">Valeur</p> } en rapport avc la gestion d'etat
